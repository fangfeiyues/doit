#### 1.http1.0，http1.1 & http2有什么区别
http1.0两个问题：
    1.连接无法复用 
    2.队首请求阻塞
http1.1 
    1.TCP通道长连接 
    2.新增首部字段Host 
    3.新增Connection默认开启Keep-Alive 
    4.新增http流水线 http pipelining可以在http端同时发起多个请求但还是回收数据还是按照响应的一一接受
http2.0
    1.Multiplexing pipelining把多个HTTP请求发送到同一个TCP连接中一一发送 在发送过程中不用等待结果
    2.服务器推送* 
    3.二进制传输
    4.头部压缩
